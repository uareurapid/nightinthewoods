First Ground - Arctic 1.0
- - -- - - -  - - - -  - - -- - --  

	Create amazing Arctic Winter scene with handpainted design.
	
	This package contains materials, textures, demo scene which helps you quickly create an amazing Arctic and winter scene.
	
	Works with Unity's Terrain painting tools.

	Features:
	
		• Colorful handpainted textures
		• 5 ice textures.
		• 3 snow textures.
		• 2 soil textures.
		• 2 rock textures.
		• Fog and show particles.
		• Mobile friendly.

	Unity and Unity Pro compatible
